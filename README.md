## 📖 О пројекту

**Анаграм** је интерактивна едукативна игра у којој деца слажу промешана ћирилична слова да погоде тачну реч. Игра је дизајнирана за ученике основних школа (1–4. разред) и покрива:

- 🐾 **Животиње** — кратке и дуже речи
- 👪 **Породица** — мама, тата, баба, деда...
- 🏫 **Школа** — оловка, књига, табла...
- 👨‍🍳 **Занимања** — лекар, пекар, учитељ...
- 🗺️ **Географија** — Београд, Дунав, Златибор...
- ⚽ **Спорт, наука, историја** и још много тога!

## ✨ Карактеристике

| Функција | Опис |
|---|---|
| **4 разреда** | Прогресивна тежина од 1. до 4. разреда |
| **370 речи** | Огромна база ћириличних речи по категоријама |
| **73 слике** | AI-генерисане илустрације за визуелно учење |
| **Емоџи подршка** | Свака реч има одговарајући емоџи |
| **Drag & Drop** | Превлачење слова на тачну позицију |
| **Помоћ (💡)** | Хинт систем за лакше учење |
| **Звучни ефекти** | Web Speech API за изговор речи |
| **Тема 🌙/☀️** | Дневна и ноћна тема |
| **Слике 📷** | Опциони приказ илустрација |
| **Стрик 🔥** | Бонус поени за узастопне тачне одговоре |
| **Конфети 🎊** | Анимирана прослава на крају нивоа |
| **Прогрес бар** | Праћење напретка кроз ниво |

## 🖥️ Платформе

### Windows (Portable EXE)
- **Фајл:** `Anagram.exe`
- **Величина:** ~100 MB
- **Тип:** Portable — не захтева инсталацију
- **Технологија:** Electron 35

### Android (APK)
- **Фајл:** `Anagram-debug.apk`
- **Величина:** ~30 MB
- **Тип:** Debug APK — инсталирај директно

## 🏗️ Изградња из извора

### Предуслови
- Node.js 18+
- npm

### Инсталација
```bash
npm install
```

### Покретање (development)
```bash
npm start
```

### Изградња Windows EXE
```bash
npm run build
```

### Изградња Android APK
Захтева Capacitor или сличан алат за паковање веб апликације.

## 🔐 Верификација (Checksums)

За потврду интегритета фајлова, упоредите SHA256 хешеве:

```
SHA256 (Anagram.exe)       = 80084D6F469954FAD883B39763409B6E3C12C6735464349C41124FE21F67845E
SHA256 (Anagram-debug.apk) = 0A13D552AC83A6E3FA224A1DDAC7E23B962D193454AD08EED999A5FD0B35C742
```

### Верификација на Windows (PowerShell)
```powershell
(Get-FileHash ".\Anagram.exe" -Algorithm SHA256).Hash
(Get-FileHash ".\Anagram-debug.apk" -Algorithm SHA256).Hash
```
